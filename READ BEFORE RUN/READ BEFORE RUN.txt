## ⚠️ Yêu cầu trước khi chạy chương trình

### Cài đặt Font chữ
Giao diện chương trình sử dụng font **Minecraft** và **Minecraft-1**.  
Nếu không cài font, giao diện sẽ hiển thị sai chữ / bị vỡ layout.

**Cách cài (Windows):**
1. Mở thư mục `Fonts/` trong thư mục dự án
2. Nhấn **Ctrl + A** để chọn tất cả 2 file font
3. Chuột phải → chọn **"Install"** (hoặc "Install for all users")
4. Khởi động lại Visual Studio nếu đang mở

**Cách cài (macOS):**
1. Mở thư mục `Fonts/` trong thư mục dự án
2. Double-click từng file font
3. Chọn **"Install Font"** ở cửa sổ Font Book
4. Khởi động lại Visual Studio nếu đang mở
